#+TITLE: Dotfiles

These dotfiles are tailored for my preferences running Omarchy.
Multiple aspects of these configs build on top on the omarchy defaults, hence these will not work well on any other distro without major changes.

** Contents

This repository contains:
- Package installation scripts, managed through =omarchy-packages/=
- Configuration files for =nix=, =hypr=, and other applications
- Personal shell scripts in =bin/=
- A unified =dotfiles= management script for syncing configs and installing packages

** Fresh Install

Run these once, in order, on a new machine. Steps 1 and 5 (=prepare= /
=finalize=) are only needed on a fresh install; steps 3 and 4 are the
day-to-day commands (see [[*Usage][Usage]]).

1. Clone this repository to =~/git/dotfiles=:
  #+begin_src bash
  git clone https://github.com/ThonyPrice/dotfiles.git ~/git/dotfiles
  cd ~/git/dotfiles
  #+end_src

2. Remove the Omarchy default configs we replace, so Stow can create clean
  symlinks instead of partially folding into existing directories:
  #+begin_src bash
  ./bin/dotfiles-prepare
  #+end_src

3. Install packages (this also installs =stow=, which the next step needs):
  #+begin_src bash
  ./bin/dotfiles packages install
  #+end_src

4. Sync configurations (symlinks every entry in =configs.list= via Stow):
  #+begin_src bash
  ./bin/dotfiles config sync
  #+end_src

5. Finalize — add =bin/= to =PATH= and install Doom Emacs:
  #+begin_src bash
  ./bin/dotfiles-finalize
  #+end_src

** Usage

After a fresh install, =dotfiles= is on =PATH= (via =dotfiles-finalize=, which
runs =stow-bin=), so it is executable from anywhere. Its purpose is to make it
*simple keeping dotfiles up to date with system configurations*:

#+begin_src bash
dotfiles config sync          # re-apply config symlinks (idempotent)
dotfiles config add <name>    # scaffold a new config to track
dotfiles packages install     # (re)install all packages
dotfiles packages add <name>  # scaffold a new package install script
#+end_src

Run =dotfiles --help= to see all available commands.

** Tmux Cheatsheet

Prefix key is =Ctrl+Space=. Press prefix, release, then press the command key.

*** Sessions (workspaces)

| Command                       | Description                  |
|-------------------------------+------------------------------|
| =tmux=                        | Start new session            |
| =tmux new -s name=            | Start new session with name  |
| =tmux ls=                     | List sessions                |
| =tmux attach -t name=         | Attach to session            |
| =tmux kill-session -t name=   | Kill session                 |
| =prefix + d=                  | Detach from session          |
| =prefix + $=                  | Rename session               |
| =prefix + s=                  | Switch session (interactive) |

*** Windows (tabs)

| Command        | Description           |
|----------------+-----------------------|
| =prefix + c=   | Create new window     |
| =prefix + ,=   | Rename window         |
| =prefix + n=   | Next window           |
| =prefix + p=   | Previous window       |
| =prefix + 1-9= | Go to window number   |
| =prefix + &=   | Kill window           |
| =Alt+L=        | Next window (custom)  |
| =Alt+H=        | Prev window (custom)  |

*** Panes (splits)

| Command             | Description                    |
|---------------------+--------------------------------|
| =prefix + "=        | Split horizontally             |
| =prefix + %=        | Split vertically               |
| =Ctrl + h/j/k/l=    | Navigate panes (vim-style)     |
| =prefix + x=        | Kill pane                      |
| =prefix + z=        | Toggle pane zoom (fullscreen)  |
| =prefix + !=        | Convert pane to window         |

*** Resize mode (custom)

| Command      | Description                          |
|--------------+--------------------------------------|
| =prefix + r= | Enter resize mode                    |
| =h/j/k/l=    | Resize pane left/down/up/right       |
| =Escape=     | Exit resize mode                     |

*** Copy mode (vim-style)

| Command      | Description              |
|--------------+--------------------------|
| =prefix + [= | Enter copy mode          |
| =v=          | Start selection          |
| =y=          | Copy selection           |
| =q=          | Exit copy mode           |

*** Other

| Command          | Description       |
|------------------+-------------------|
| =prefix + I=     | Install plugins   |
| =prefix + R=     | Reload config     |
| =prefix + ?=     | List keybindings  |
